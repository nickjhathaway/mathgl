Стабильная версия

Вы можете загрузить текущую версию MathGL в следующих конфигурациях:
  • исходные тексты.
  • Win32 бинарники для MinGW (i686, GPL версия)
  • Win32 бинарники MinGW (i686, LGPL версия -- нет поддержки GSL и HDF5)
  • DevPak для Dev-C++ или Code::Blocks (GPL версия)
  • RPM для Fedora
  • DEB для Debian Sid
  • PDF файл документации на английском
  • HTML документация на английском
  • HTML документация на русском
  • PDF презентация основных возможностей

Более старые версии библиотеки можнозагрузить со страницы проекта на sourceforge.

Также можно отдельно загрузить интерпретатор скпритов MGL interpreter:

  • Win32 бинарники - архив с утилитами mgl2png.exe, mgl2eps.exe и т.д.
  • Исходные тексты включены в библиотеку тут.
Note, that GSL and PNG libraries should be installed in system. Windows binary of MathGL library (from v.1.9) include all external libraries as statically linked in libmgl.dll. So, you don't need to install any additional packages if you want to use only MathGL features.

Дополнительный набор файлов шрифтов для MathGL с различными начертаниями. Отмечу, что набор символов (glyphs) в них может быть меньше чем в шрифте по умолчанию. Это приведет к тому, что не все возможные символы TeX будут отображены.

  • STIX – шрифт по умолчанию для MathGL.
  • Adventor – TeX Gyre шрифт на основе URW Gothic L family (типа Avant Garde Gothic).
  • Bonum – TeX Gyre шрифт с засечками на основе URW Bookman L family.
  • Chorus – TeX Gyre шрифт на основе URW Chancery L Medium Italic.
  • Cursor – TeX Gyre моноширинный шрифт с засечками на основе URW Nimbus Mono L (типа Courier).
  • Heros – TeX Gyre шрифт на основе URW Nimbus Sans L (типа Helvetica).
  • HerosCN – "сжатая" версия предыдущего.
  • Pagella – TeX Gyre шрифт с засечками на основе URW Palladio L (типа Palatino).
  • Schola – TeX Gyre шрифт с засечками на основе URW Century Schoolbook L.
  • Termes – TeX Gyre шрифт с засечками на основе Nimbus Roman No9 L (типа Times).
